Comando: git --version
Definición breve: Muestra la versión instalada de Git
Ejemplo de uso: git version 2.54.0.windows.1

Comando: git init
Definición breve: Inicializa un repositorio en la carpeta actual (y queda invisible)
Ejemplo de uso: git init
"Initialized empty Git repository in C:/Berta/CURSO - Fullstack Javascript/Proyectos/Javascript/Ejercicios_practico/.git/"                                                                                                               Initialized empty Git repository in C:/Berta/CURSO - Fullstack Javascript/Proyectos/Javascript/Ejercicios_practico/.git/

Comando: git add
Definición breve: Agrega cambios al área de preparación (staging)
Ejemplo de uso: git add . (agrega todos los archivos)

Comando: git commit
Definición breve: Registra los cambios con un mensaje descriptivo y confirmo la instrucción anterior.
Ejemplo de uso: git commit -m "Primer commit"
